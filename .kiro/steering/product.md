# Product Overview

This is an MCP (Model Context Protocol) configuration project for Kiro AI assistant integration with Google Sheets.

## Purpose
- Provides MCP server configuration for Google Sheets integration
- Enables AI assistant to interact with Google Sheets through the mcp-gsheets package
- Part of the Kiro AI ecosystem for enhanced spreadsheet automation

## Key Features
- Google Sheets read/write operations via MCP
- Authenticated access using Google service account credentials
- Seamless integration with Kiro AI assistant workflows
